# iStorage4SME web app

Built using Nuxt v3
Using Sanity.io as headless CMS

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```

## Production

Generate the SSG for production:

```bash
npm run generate
```
